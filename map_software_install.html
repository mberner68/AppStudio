Map Software Installation

Home

  1. Click on link to editor
  2. You will see a default map to replace map with Support Group Map do the following:
    1. Click New (state_test for the name)
    2. Click Create
    3. Download Spec File
    4. Click Import
    5. Click Save
  3. You will see a default map and to replace map with State Honor Roll do the following:
    1. Click New (test for the name)
    2. Click Create
    3. Download Spec File
    4. Click Import
    5. Click Save
  4. You will see a default map and to replace map with a new one do the following:
    1. Click New (Add a name)
    2. Click Create
    3. Add a Map by clickin on Image Settings > Image (add a new Url)
    4. Click Save

IMPORTANT

Please note all data is saved locally in your browser of your computer, to save the data follow these steps:

  1. Click save after finishing
  2. Click Export data to save a hard copy
  3. Hard Copy can be used later to import
  4. Next you use the software should appear whn you click load
  5. Do not clear appilcation data on your browser or you will lose your map
